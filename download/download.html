Installation Instructions

Step 1: Extract the Files

After downloading, extract the contents:

tar -xzf notes-vibes-app-bundle.tar.gz
tar -xzf notes-vibes-app.tar.gz
cd notes-vibes-app

Step 2: Run the Setup Script

Run the interactive setup script:

node setup.js

Step 3: Configure Environment

Edit the .env file with your database credentials and Spotify API keys.

Step 4: Start the Application

npm run dev

Username: demo
Password: password

System Requirements

  • Node.js 18 or later
  • PostgreSQL database
  • npm (included with Node.js)
